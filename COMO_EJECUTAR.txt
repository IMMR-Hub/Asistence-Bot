¿QUE PASA SI ME DA ERROR?
═════════════════════════

Error: "Docker no está instalado"
→ Descarga Docker desde: https://www.docker.com/products/docker-desktop

Error: "No se encuentra docker compose"
→ Docker Compose debería venir con Docker
→ Intenta reinstalar Docker

Error: "Puerto 8000 ya está en uso"
→ Algo más está usando ese puerto
→ Intenta reiniciar la computadora

Error: Otra cosa
→ Toma captura de pantalla del error
→ Mándamela y te ayudo
